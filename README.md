# Sate-Match
Game PC sederhana drag and drop untuk tugas UAS Grafkom Kelompok 1

Sate Match adalah game di mana pemain berperan sebagai penjual sate.  
Kamu harus menyiapkan sate sesuai pesanan pelanggan, tapi setiap sate memiliki waktu matang berbeda, dan setiap pelanggan memiliki waktu tunggu berbeda.

## Cara Main
- Drag and drop sate sesuai pesanan pelanggan.
- Perhatikan waktu matang sate dan waktu tunggu pelanggan.
- Semakin cepat dan tepat, semakin tinggi skor.

## Cara Menjalankan
Game dapat kamu mainkan langsung online di Itch.io: [https://sachi079.itch.io/sate-match](https://sachi079.itch.io/sate-match)

## Tools yang Digunakan
- **Blender** 
- **IbisPaint** 
- **FireAlpaca**
- **Construct 2** → mengolah semua asset menjadi sprite, membuat logic sederhana di event sheet, dan membangun game

## Langkah Singkat Pembuatan Game
1. Buat semua asset yang diperlukan menggunakan Blender, IbisPaint, dan FireAlpaca.  
2. Kumpulkan semua asset yang sudah jadi.  
3. Masukkan asset ke Construct 2 → ubah menjadi sprite, animasi, dan logikakan di **event sheet**.  
4. Export game ke HTML5
